# django-red

## linux安裝方式:
- 環境 (debian/ubuntu):
    ```bash
    git clone https://github.com/Michaelrobot/django-red.git
    cd django-red
    docker login
    docker-compose pull
    docker-compose up
    ```
## 執行
- 環境 (debian/ubuntu):
    ```bash
    docker-compose up
    ```
